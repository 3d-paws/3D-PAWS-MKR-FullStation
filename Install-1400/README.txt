Install Instructions

Exit Arduino Serial Monitor if it is running

Push the reset button twice quickly on the Feather board to set firmware download mode.
   The red LED by the battery connector will pulse quickly.
   COM port will change when in firmware download mode. 

Run install.bat

   Windows Defender may prevent the install.bat from running. 
      Open the properties of the install.bat file and see if there is a "unblock" check box. If so, select it.
                  
   Enter the COM port. - More COM information can be found with Device Manager
   Wait. A lot of read/write messages will scroll then you will see percentage download information

Exit the script
